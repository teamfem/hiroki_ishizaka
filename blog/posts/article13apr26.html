FEniCSxをGoogle Colab上で動かし，半線形楕円問題のP1有限要素近似，相対誤差，2D/3D可視化を確認するまでの流れと，最終コードの1行ごとの解説をまとめました．

0. Introduction

このページでは，FEniCSxで書かれた主問題のP1有限要素近似をGoogle Colab上で実行できるように整理した流れをまとめます．さらに，最終的に使ったPythonスクリプトを全文掲載し，1行ごとに丁寧な解説も付けます(ChatGPTにお願いしています)．なお，コード作成には，ChatGPTを補助として使っています．ChatGPTによるコードの正確性も見ていきます．今回のコード作成も最初のプロンプト段階では不十分でした．良いコードを得るには，複数回の良いプロンプトを書く必要があります．今回は初回ブログなので，とにかく"書く"を意識しています．よって，正確性は犠牲にしています．次回から，簡単な問題から詳細に見ていきます．

対象とする問題は \[ -\Delta u + u^3 = f \quad \text{in } \Omega=(0,1)^2,\quad u=0 \quad \text{on } \partial\Omega \] です．厳密解として \[ u(x,y)=\sin(\pi x)\sin(\pi y) \] を用い， \[ f=2\pi^2u+u^3 \] を右辺として与えます．この設定により，相対誤差 \[ \displaystyle \frac{|u-u_h|_{H^1}}{|u|_{H^1}},\quad \frac{\|u-u_h\|_{L^2}}{\|u\|_{L^2}} \] を確かめます．

1. Google Colabでの実行手順

ここでは，最終版スクリプト extract_fem_primal_with_relative_error_checks_and_plots_colab_fix4_3d.py をColabで動かすときの流れを，セルごとに書いておきます．

1.1 FEniCSxの導入

最初のセルでは，Colab上にFEniCSxを入れます．すでに入っていればそのままimportし，入っていなければインストールスクリプトを走らせます． 

try:
    import dolfinx
except ImportError:
    !wget "https://fem-on-colab.github.io/releases/fenicsx-install-release-real.sh" -O "/tmp/fenicsx-install.sh"
    !bash "/tmp/fenicsx-install.sh"
